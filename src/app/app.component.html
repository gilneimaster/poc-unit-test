Este é o AppComponent

  • O AppComponent é carregado a partir da rota raiz do AppModule e servirá apenas para testes locais do módulo que será exposto.
  • O AppModule é o módulo principal do projeto. Ele não deverá ser exposto como um módulo de micro front-end.
  • Para este projeto, foi criado e configurado o PocUnitTestModule como módulo de micro front-end exposto e está pronto para ser carregado a partir de um módulo shell
